Tentang Web

Hello World!, selamat datang di ce i dojo, kitab saktinya kode igniter.

Disini kamu bisa belajar tentang Framework PHP yang cukup terkenal, yaitu si Codeigniter.
Dilengkapi dengan dokumentasi berbahasa Indonesia, diringkas secara singkat, namun padat dan jelas.

Materi akan disajikan berurutan per bab, dibahas mulai dari yang dasar dulu, sampai ke tingkat lanjut. Mulai dari konfigurasi CI, konsep MVC (Model View Controller), CRUD database, hingga library akan dibahas disini.

Nggak cuma CI lo!, disini juga membahas tentang programming Web dasar dengan HTML, CSS, JS, dan framework - framework frontend yang terkenal.

Semua konten CI Dojo bersifat Open Source, dapat digunakan secara bebas oleh publik.
